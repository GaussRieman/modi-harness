# V0.6.c Canonical Memory Scopes — Implementation Plan

## Goal

Remove legacy Memory scope names and make the runtime, examples, tests, and
current docs use only:

```text
user, workspace, agent, thread
```

V0.6.c is a breaking cleanup. It does not read or write
`memory/project` or `memory/conversation`.
