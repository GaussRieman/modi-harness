# V0.6.b Core Concepts Alignment — Implementation Plan

## Goal

Align the docs around eight first-principles concepts:

```text
Workspace, Session, Thread, Run, Store, Context, Memory, Trace
```

This release aligns documentation and adds minimal runtime compatibility for
the preferred Memory scope names. It must not rename public APIs or change
storage layout.
