# V0.6.e Execution Efficiency — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove deterministic extra turns from the runtime by executing all tool calls proposed in one model turn and caching per-run memory recall/selection until a committed memory write invalidates it.

**Architecture:** Keep the existing graph topology unchanged. Change only the execution substrate:

- `execute_tool_node` processes the whole `pending_tool_calls` list serially in one node visit, preserving one `tool_result` per model `tool_use`.
- `model_turn_node` wraps the existing memory recall/select pair with an in-process `RunRecallCache` stored on `GraphDeps`.
- `execute_tool_node` invalidates the recall cache when existing memory-write detection says a write committed.

**Tech Stack:** Python 3.12, pytest, LangGraph, modi_harness (`src/modi_harness`), uv.

**Spec:** `docs/superpowers/specs/2026-06-12-v0.6e-execution-efficiency-design.md`

**Explicitly out of scope:** concurrent tool execution, graph topology changes, fixed plan→execute orchestration, structured-output hardening, preflight checks, token/cost observability, LRU/TTL cache eviction.

Implementation sketch:

```python
def execute_tool_node(state, config):
    pending = state.get("pending_tool_calls") or []
    if not pending:
        return {}

    update = _empty_tool_batch_update()
    for index, proposal in enumerate(pending):
        if proposal.get("malformed"):
            _merge_malformed_result(...)
            continue
        dispatch = deps.tools.execute_tool_call(...)
        if dispatch.outcome == "interrupt" and dispatch.decision is not None:
            return _handle_batch_interrupt(...)
        _append_dispatch_result(update, state, deps, dispatch)
    update["pending_tool_calls"] = []
    return update
```

Use helper functions only where they keep the node readable. Keep edits localized to `nodes.py`.
